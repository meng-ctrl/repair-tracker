# 维修工作记录 App (RepairTracker)

## 项目结构
```
repair-tracker/
├── index.html          # 主页面
├── manifest.json       # PWA 配置
├── sw.js               # Service Worker 离线支持
├── app.js              # 主逻辑
├── styles.css          # 样式
└── icons/              # 图标目录
```

## 功能清单
- [x] 维修单列表（按时间排序，月度折叠）
- [x] 流程进度展示（时间排序，图片/视频/文字）
- [x] 预设流程标题
- [x] 权限管理（管理员/完全授权/部分授权）
- [x] 分享维修单
- [x] NAS 云同步（可选）
